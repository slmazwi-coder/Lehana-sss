# Lehana Senior Secondary School

Official website for **Lehana Senior Secondary School**, Castle Rock, Tsekong Location (Ward 8), Mount Fletcher, 4770, Eastern Cape.

- **EMIS Number:** 200501358
- **Established:** 1 January 1957
- **Quintile:** Q1 (No-Fee School)
- **Principal:** Mrs Nyali
- **District:** Joe Gqabi, Elundini Local Municipality

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   ```
   npm run dev
   ```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React icons
- Google Gemini AI (chatbot)

## Features

- Responsive school website with admissions, boarding, staff, achievements, sports, and contact pages
- AI-powered chatbot assistant (Gemini)
- Online application forms (General & Boarding)
- Student and Admin portals
- Multilingual support (English, isiZulu, isiXhosa, Sesotho)
